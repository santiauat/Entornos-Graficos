Consulta a una base de datos: Para comenzar la comunicación con un servidor de base de datos MySQL, es necesario abrir una conexión a ese servidor. Para inicializar esta conexión, PHP ofrece la función 
    
    rta: mysqli_connect (mysql_connect)

Todos sus parámetros son opcionales, pero hay tres de ellos que generalmente son necesarios:
    rta:
         - $hostname
         - $nombreUsuario
         - $contraseña

Una vez abierta la conexión, se debe seleccionar una base de datos para su uso, mediante la función
    rta: mysqli_select_db (mysql_select_db) 

Esta función debe pasar como parámetro 
    rta: 
        - $nombreConexión, se obtiene mediante mysqli_connect
        - $nombreBaseDatos

La función mysqli_query () se utiliza para  
    rta: ejecutar una consulta a la base de datos que especifiquemos.

y requiere como parámetros 
    rta: 
        - $nombreConexion
        - $query

La cláusula or die() se utiliza para
    rta: Capturar el error y mostrar un mensaje.

y la función mysqli_error () se puede usar para
    rta: Devuelve el último mensaje de error para la llamada más reciente a una función de MySQLi que puede haberse ejecutado correctamente o haber fallado.
